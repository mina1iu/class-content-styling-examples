FontAwesome Icons

How To:

  1. Go to fontawesome.com/start - sign up and get the kit
  2. Add the kit JS before the closing body tag (like any other JS file)
    <script src="https://kit.fontawesome.com/GETYOUROWN.js" crossorigin="anonymous"></script>
    or (for now) use the CDN link to their CSS file:
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
  3. Pick an icon. For example the LinkedIn icon from brand icons and gets its code snippet
    example: <i class="fa-brands fa-linkedin"></i>
  4. style (size, color, border, ...) the icon like text
    see fa-icons.css for an example
  5. Enjoy!
    Remember to cite all the CSS, JS, and HTML
